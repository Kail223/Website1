Stock Prediction Tool

This tool predicts stock market trends with over 60% accuracy using advanced machine learning algorithms.

Project Overview

    Yahoo's public top 100 stocks were gathered and called within the application to serve as the data bank.

    Matplot and Pandas were used to plot the data within the page, and display info in the form of a graph.

    Facebook prophet was utilized to create predictions based on the previous data and plot possible trends lines for up to 4 years.
