Mobile Development Internship

Flutter App Development: 2024

Project Overview

  • The main objective was to create a mobile hub from the ground up, that had a secure login system.
  • The app needed to be able to host all of the required student functionality, and host video calls for the classrooms.
  • The app also had to be compatible with the standalone "mute" device and both Android and IOS.

Key Development Aspects

  • Flutter needed to be used to create a login system, and a hub style home page with access to everything within a few button presses.
  • The video functionality specifically needed to use jitsi while being integrated within the app.
  • The app also needed to have a student view, and a teacher view with more privileges and functionality.
